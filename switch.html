Switch v C



Příkazy switch pomáhají řídit složité podmíněné operace a operace case větvení. Příkaz switch přenese ovládací prvek na příkaz v jeho těle.

switch ( expression ) statement

labeled-statement:

case constant-expression : statement

default : statement
